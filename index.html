Crazy Santa

A Python Game

A Christmas themed game similar to the Helicopter game where the user plays as Santa Claus riding on a flying sleigh, and tries to navigate a field of obstacles floating through the air. The user can hold the ‘Up’ arrow to fly upwards and release to let the sleigh fall down. If you are playing for the first time or are just having some trouble, there is a beginner mode which calculates and displays the optimal path through the obstacles. If you are looking for a challenge, try the arcade mode. There are now presents to be collected and moving obstacles and poison to be avoided. Written in Python and Tkinter.
