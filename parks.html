In Spring 2015, I worked on a group project where a group of Informations Systems students and I served as technical consultants for a non profit organization. Our client was the Pittsburgh Parks Conservancy. I served mainly as the Designer and Android Developer in this project.

Problem

Our main goal for the project was to identify technical problems in their organization and work with our client to design, build, and deliver an information system solution to meet our client’s needs.

After talking to our client, we found out that the main problem in the organization is that the Pittsburgh Parks Conservancy (PPC) has an Android and iOS application, but the current UI of the application is overly complicated and makes it very hard to use.

Our goal for the project was design and develop a new application for the PPC.

User Stories

We interviewed our clients and some potential end users of the product and created the following user stories to include in the application.
  • As a park visitor I want to see a map of the park so that I can navigate and discover new park features based on my location
  • As a park visitor I want to discover new points of interest (such as fountains, parking lots, or bathrooms) so that I can more easily navigate the park
  • As a park visitor I want to see park events so that I can attend events that interest me
  • As a park visitor I want to donate to PPC so that I can support their work
  • As a park visitor I want to keep in touch with the PPC on social media so that I can receive news about the parks

Design Process

We took the main, essential user stories and converted it into functionalities. The design has its main functionalities in different tabs so the user can easily navigate from functionality to functionality.

This was the rough draft of our idea.




In order to reduce the number of clicks for a user to get to a certain functionality, we’ve decided to show the list of attractions along with their categories in one page rather than requiring the user to click an extra time to get to a specific attraction.

We also included a short description and a photo gallery of the park.
