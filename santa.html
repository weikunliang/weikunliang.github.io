Project Description

This was my final project for my Fundamentals of Programming class. I was inspired by the upcoming Christmas season so I decided to make a Christmas themed Helicopter game where instead of a helicopter, we have Santa Claus riding on a flying sleigh. The user plays as Santa Claus and the goal of this game is try to navigate a field of obstacles.There are 2 modes of the game - normal and arcade mode. Written in Python. The code can be found here.
